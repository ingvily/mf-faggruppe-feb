# mf-faggruppe-feb

Start `jupyter notebook`



Innstaller følgende for å kjøre notebooken lokalt: 


Python versjon 2.7 

`brew install python`

eller 3.3: 

`brew install python3`

Jupyter: 

`pip install jupyter`

Numpy:

`pip install numpy`

Pandas

`pip install pandas`

Scipy

`pip install scipy`

Scikit-learn (versjon 0.18): 

`pip install -U scikit-learn`
